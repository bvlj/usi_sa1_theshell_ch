--- layout: page category-page: fs category-title: FileSystem tags: directory list move rename author: Mattia Hijman title: mv --- The mv command is used to move a file from a given path to another. It can also be used to rename a file by moving it into the same directory, but with a different name.
It stands for MoVe. 
mv [-finv] source target

Flags

  • -f: Do not prompt for confirmation before overwriting the destination path.
  • -i: Asks user confirmation before proceeding.
  • -n: Do not overwrite an existing file.
  • -v: Cause mv to be verbose, showing files after they are moved.
