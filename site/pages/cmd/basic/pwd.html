The pwd command (path of working directory)
The absolute pathname of the current working directory is given as result.

  cd /home/user
  pwd = /home/user
  • -P Show the working directory path without symbolic link (links to folder)
    Example: PWD /home/symphoto, symphotos is a symlink to /home/photos PDW-P will show /home/photos
  • -L Show the working directory logical path with symbolic link.
    Example: There is a symbolic link /home/symphoto, that redirect to /home/photo, PWD -L will show /home/symphoto
  • Without flag, the -L is assumed, so the normal PWD is in reality PDW-L
