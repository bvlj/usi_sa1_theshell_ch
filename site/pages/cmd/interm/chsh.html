The chsh command cheanges user's login shell property:

  • chenage Password of the user
  • shell local directory
  • Full name of the user
  • Office location
  • Office phone
  • home phone

      • Example: for cheanging password after the command chsh use passwd.

      chsh [-flag] user
      • -s (shell)set shell as the login shell
      • -l (listChanges the symbolic link and not the file or directory referred by the symbolic link.
      • -R changing the ownership for each file in a folder, if encounter symbolic link it will cheange the owenership of that symbolic link, and the directoryfile referred in the symbolic link, but the directory is not further transversed.
      • -RH changing the ownership for each file in a folder, if encounter symbolic link it will cheange the owenership of that symbolic link, the directory/file referred in the symbolic link, and all the directory is further transversed.
      • -RL If the -R option is specified and a symbolic link referencing a file of type directory is specified on the command line or encountered during the traversal of a file hierarchy, the chown command shall change the user ID (and group ID, if specified) of the directory referenced by the symbolic link and all files in the file hierarchy below it.
      • -RP changing the ownership for each file in a folder, if encounter symbolic link it will cheange the owenership of that symbolic link, but will not chenage the the directory/file referred in the symbolic link, and the directory is not further transversed.
