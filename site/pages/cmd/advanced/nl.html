The nl is a command that permits to number the lines. Through some flags you can decide how to filter this command.

We have a file named example.txt that contains 5 elements, to test this command: {% highlight ruby linenos %} Car Computer Robot Smartphone Videogame {% endhighlight %} The syntax command is: 
 nl [flags][file]
{% highlight ruby linenos %} nl example.txt 1 Car 2 Computer 3 Robot 4 Smartphone 5 Videogame {% endhighlight %}

How you can see the command has numbered the lines.

Flags

Here a list that contains the main flags of these command:

  • -b: Specify the lines to be numered
    • {% highlight ruby linenos %} nl -b p^[cv] example.txt 1 Car 2 Computer 3 Videogame Robot Smartphone {% endhighlight %}

    p^[cv] says to nl to number only the lines that start with c and v.

  • -bn: The flag n goes with b, doesn't number any lines
    • {% highlight ruby linenos %} nl -bn example.txt Car Computer Videogame Robot Smartphone {% endhighlight %}
  • -s: Usually nl separes the number of the line from the text with a tab, with -s flag you can choose another separator, in this example we will use "=".
    • {% highlight ruby linenos %} nl -s= example.txt 1=Car 2=Computer 3=Videogame 4=Robot 5=Smartphone {% endhighlight %}
