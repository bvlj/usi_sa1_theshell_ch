--- layout: about category-page: about category-title: About tags: website move next back author: Joey Bevilacqua title: Navigation --- You can navigate in our website with your keyboard using some tricks:
Key Function
~ In a shell, the "~" character represents home. So it does on TheShell
w Brings you back to the top of the page
a Brings you to the previous page (if any)
d Brings you to the next page (if any)
s Brings you to the topic page of the page you're currently reading
k Brings you back to the top of the page. This is the same as vi up key command
h Brings you to the previous page (if any). This is the same as vi left key command
l Brings you to the next page (if any). This is the same as vi right key command.
s Brings you to the topic page of the page you're currently reading. This is the same as vi down command.
0 Brings you to the home page
1 Brings you to the basic commands topic page
2 Brings you to the intermediate commands topic page
3 Brings you to the advanced commands topic page
4 Brings you to the file system topic page
5 Brings you to the scripting topic page
6 Brings you to the about topic page
The navigation shortcuts will not be available when you're doing a search through the search bar.
